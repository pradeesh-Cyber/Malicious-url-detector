// Process

How ThreatScan Works

From raw URL to threat verdict in milliseconds. No external lookups. Pure lexical analysis.

01

Submit URL or CSV

Paste any URL into the analyzer or upload a CSV file containing up to 5,000 URLs for batch processing.

02

Feature Extraction

24 lexical features are extracted: URL length, entropy, IP presence, special characters, domain depth, TLD, shortener flags, and more.

03

AI Classification

The XGBoost model — trained on 651,191 labeled URLs — classifies the URL into one of four categories with a confidence score.

04

Verdict Delivered

Receive a Genuine or Malicious verdict. If malicious, the specific attack type is identified along with probability scores for all classes.

Platform Features

Real-Time Analysis

Single URL predictions return in under 100ms. The model runs entirely server-side with no third-party lookups required.

📊

Multi-Class Classification

Detects four distinct categories: Benign, Phishing, Malware, and Defacement — not just binary safe/unsafe.

📁

Batch CSV Processing

Upload a CSV with up to 5,000 URLs and receive a complete report with per-URL verdicts, attack type counts, and summary statistics.

🎯

Confidence Scoring

Every prediction includes a confidence percentage and the full probability distribution across all four classes.

🔮

24-Feature Lexical Analysis

No DNS lookups or external APIs. ThreatScan extracts 24 features directly from the URL string — entropy, character ratios, IP detection, port detection, hex encoding, URL shortener flags, domain depth, TLD length, and more. Fast, private, and fully offline-capable.

// Threat Intelligence

Threats We Detect

Trained on a curated dataset of 651,000 URLs across four distinct attack categories.

✅

Benign

Legitimate URLs from trusted sources. No malicious indicators detected across any of the 24 extracted features.

🎣

Phishing

URLs designed to steal credentials by mimicking legitimate services via spoofed domains and obfuscation patterns.

🦠

Malware

URLs serving malicious payloads including trojans, ransomware, and spyware. Identified via lexical anomalies.

🖥

Defacement

Compromised websites with altered content, used to spread propaganda or demonstrate unauthorized access.
